Color Toy

An experiment by UberPilot.

Some colors used from Nord Theme.

Parameters

The middle color to generate. All other colors are based off of this color.

Luminance Delta is the amount of brightness changed per color; this number is used when it is less that the number of steps to take the starting color from light to dark.

The amount to hue shift each color; this number is used when it is less than the total number of steps to yellow. To disable this behavior, check in the advanced settings.

The amount that saturation factors into each luminance step. A factor of this number is used in each direction.

The number of colors to generate.

Advanced Settings

These settings give additional control over the process, but may result in undefined behavior.

Correction

By default, the generator will not go past what is set as the yellow hue when generating brigher colors, nor go past the blue hue when generating darker colors. This unlocks that limitation.

Low Values

Low Values split the behavior of the generator; the above values will be used for generating brighter colors, and the below values will be used for generating darker colors.
